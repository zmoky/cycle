# tron

TRON like web based multiplayer game

## Install

1. Install node.js version 10 or better

2. Install node global dependencies

```
npm i -g typescript tslint
```

3. Install local node dependencies

```
npm i
```

## Development

Start in watch mode

```
npm run watch
```

## Production

Start in live mode

```
npm run build
npm run live
```
